# notekeeper-cli

A tiny, dependency-free command-line note manager in Python.

## Features
- Add a note with a title and body
- List note titles
- View a note
- Delete a note
- Data stored locally in `~/.notekeeper/notes.json`

## Requirements
- Python 3.8+

## Installation
Clone the repository and make the main script executable:

```bash
git clone https://github.com/<your-username>/notekeeper-cli.git
cd notekeeper-cli
# optional: create virtualenv
chmod +x notekeeper.py
